## build and tag

```
$ git clone...
$ npm install
$ npm run build
$ git add -f dist
$ git commit -m "added dist folder"
$ git push
$ git tag v2.0.8-forked
$ git push origin master
$ git push origin v2.0.8-forked
```

## Install

```
$ npm install github:roip-inspekto/sockette#v2.0.7-forked
```
